# @wuxh/utils

工具集

## Install

```bash
npm i @wuxh/utils
# or
yarn add @wuxh/utils
```

## Browser

```html
<script src="https://unpkg.com/@wuxh/utils"></script>
<script>
  console.log(utils)
</script>
<!-- or -->
<script type="module">
  import * as utils from 'https://unpkg.com/@wuxh/utils?module'
  console.log(utils)
</script>
```
